1. 命令行创建npm的配置文件
  a. npm init
2. 添加一个gulp的依赖
  npm install gulp --save-dev
3. 在项目根目录下添加一个gulpfile.js文件，这个是gulp的主文件，这个文件名是固定的
4. 在gulpfile中抽象我们需要做的任务

## git基本操作
- 1.创建 README.md
	- echo "# 随便写点内容" >> README.md
- 2.git init
- 3.git add .
- 4.git commit -m "写点识别内容"
- 5.git remote add origin 仓库链接
- 6.git push -u origin master
	- 输入名称 密码
